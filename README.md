# 🎓 SmartPath Counselor

> To guide self-learning in ICT with smart recommendations and clear career paths.

---

## 📌 Overview
SmartPath Counselor is an AI-powered chatbot designed to help students explore the field of Information and Communication Technology (ICT). It provides structured guidance, degree recommendations, and visual explanations to support self-learning.

---

## 🚀 Features

- 💡 Intelligent ICT guidance
- 🎯 Degree & career recommendations
- 📊 Visual diagrams & explanations
- 📄 PDF/TXT knowledge support
- 💬 Chat-based interaction
- 🧠 AI-powered responses (Groq + Gemini)

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Groq API (LLaMA 3)
- Google Gemini AI
- Replicate (Image Generation)
- SQLite Database
